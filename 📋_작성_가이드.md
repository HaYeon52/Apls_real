## ⚠️ 주의사항

### 1. 과목명은 정확히!
`courseData.ts`의 과목명과 **완전히 일치**해야 합니다.

#### 자주 헷갈리는 과목명:
- ✅ **운용관리** (운영관리 아님!)
- ✅ **기계학습과데이터마이닝** (데이터마이닝과기계학습 아님!)
- ✅ **공급사슬경영(Scm)** (공급망관리 아님!)
- ✅ **객체지향프로그래밍+데이터구조론** (통합 과목)

### 2. 학기 형식
- ✅ `"1-2"`, `"2-1"`, `"2-2"`, `"3-1"`, `"3-2"`, `"4-1"`
- ❌ `"1학년 2학기"`, `"2-1학기"` 등

### 3. 필수 과목은 자동 추가됨
- **전공기초(필수)** 과목은 작성하지 않아도 자동으로 추가됩니다
- **선택 과목만** 작성하세요

### 4. 추천 개수
- 각 학기당 **1~3개** 정도가 적당합니다
- 너무 많으면 선택이 어려워집니다

## ✍️ 추천 이유 작성 팁

### 좋은 예시 ✅
```typescript
{
  courseName: "기계학습과데이터마이닝",
  reason: "데이터에서 패턴을 찾아내는 머신러닝 알고리즘을 배우며, 최근 산업 전반에서 데이터 분석 역량이 중요해짐에 따라 진로와 무관하게 데이터 리터러시를 갖추기 위해 수강하는 것이 좋습니다."
}
```

### 나쁜 예시 ❌
```typescript
{
  courseName: "기계학습과데이터마이닝",
  reason: "데이터를 배웁니다."  // 너무 추상적!
}
```
